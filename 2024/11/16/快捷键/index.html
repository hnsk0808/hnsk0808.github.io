常用快捷键

Arch Linux

BiliBili

  • Q点赞
  • W投币
  • E收藏
  • F全屏
  • D弹幕

Edge

  • Ctrl + Tab向下选择标签页
  • Ctrl + Shift + Tab向上选择标签页
  • F5刷新
  • F11全屏
  • F12调试

Steam

Terminal

Visual Studio Code

  • Ctrl + Tab切换打开的标签页
  • Ctrl + Backtick切换Terminal显示/关闭

Visual Studio

  • Ctrl + K, Ctrl + C选中代码注释
  • Ctrl + K, Ctrl + U选中代码取消注释

Windows 11

  • Alt + F4关闭当前应用
  • Alt + Shift切换输入法语言
  • Alt + Tab切换应用
  • Ctrl + Shift + Esc打开Task Manager
  • F1截图Snipaste
  • F3贴图Snipaste
  • Win + L锁屏
  • Win + S搜索
  • Win + E打开Explorer
  • Win + Tab切换桌面
  • Win + Up/Down/Left/Right修改窗口尺寸/位置
